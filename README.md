# Agenda

## A JavaScript CRUD project to study node.js

### Project dependencies: 
- connect-flash
- connect-mongo
- core-js
- css-loader
- csurf
- dotenv
- ejs
- express
- express-session
- helmet
- mongoose
- regenerator-runtime
- style-loader

### Getting Started

To download the packages run:  `npm install`

then run `npm server`

access [`localhost:8080`](http://localhost:8080) from your browser
